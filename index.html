📋 Quick start

Up in
4 Steps

1

Clone the repo

Open Termux and clone using SSH or HTTPS above.

2

Launch it

Run bash ~/rebuild/rebuild.sh — the boot sequence runs and Bojack greets you.

3

Optional — set alias

Add alias rebuild='bash ~/rebuild/rebuild.sh' to your ~/.bashrc then source ~/.bashrc. Now just type rebuild.

4

Nothing runs without you

Every module asks before it acts. You approve every change. Strategy Over Impulse — always.
